UpperCaseFirst

Easy

Given a string, return that same string with all of the upper-case characters at the front.

All other orderings should be kept. There are no spaces.

Parameter(s):

  • str: a string

Returns:

  • string: a string with all of the upper-case characters at the front

Constraints:

  • str ≠ null
  • 1 ≤ str.length ≤ 1000

Example 1:

Input:

"heLlO"

Output:

"LOhel"

Example 2:

Input:

"fOoBAr"

Output:

"OBAfor"

Example 3:

Input:

"cake"

Output:

"cake"
