Sum

Easy

Given two integers, return their sum.

Parameter(s):

  • a: first integer
  • b: second integer

Returns:

  • integer: the sum

Constraints:

  • a ≠ null
  • b ≠ null
  • -2147483647 ≤ a ≤ 2147483647
  • -2147483647 ≤ b ≤ 2147483647

Example:

Input:

1 2

Output:

3
